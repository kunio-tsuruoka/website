# CDP・データ活用コンテンツ 収録用質問リスト

## 企画概要

- **目的**: 経営者・発注担当者向けに「データ活用って結局何ができるの？」を噛み砕いて伝えるコンテンツ
- **話者**: データサイエンティスト（佐藤さん）
- **ターゲット読者**: 非エンジニアの経営者・事業責任者（「データ活用したいが何から始めればいいかわからない」層）
- **形式案**: インタビュー記事 or 対談記事（社長×データサイエンティスト）

## 質問リスト

### パート1: 導入（経営者の不安に寄り添う）

**Q1. 「データ活用」って言葉をよく聞きますが、結局のところ経営者にとって何が変わるんですか？**
→ 狙い: 抽象的な「データ活用」を、売上・コスト・意思決定の具体的な変化に落とし込む

**Q2. 正直なところ、中小企業でもデータ活用って意味ありますか？大企業向けのイメージがあるんですが。**
→ 狙い: 「うちには関係ない」という経営者の心理的ハードルを下げる。BigQueryの無料枠やGA4の話を引き出す

**Q3. 「データはあるけど活用できていない」という企業が多いと思います。なぜそうなるんでしょう？**
→ 狙い: よくある失敗パターンを具体的に挙げてもらう（Excel管理の限界、部署間のデータ分断など）

### パート2: CDPの具体的な価値（「何ができるか」を経営者目線で）

**Q4. CDPを入れると、具体的にどんなことがわかるようになりますか？経営判断にどう使えますか？**
→ 狙い: 技術の話ではなく「経営にとっての価値」を語ってもらう
→ 例: 「この商品を買った人は3ヶ月以内にリピートする確率が高い」→ 施策に直結

**Q5. 「優良顧客を見つける」とよく言いますが、具体的にどうやって見つけるんですか？素人にもわかるように教えてください。**
→ 狙い: RFM分析を平易に解説。「最近買った」「よく買う」「たくさん買う」の3軸で分ける、という話
→ SEOキーワード: RFM分析 やり方

**Q6. LTV（顧客生涯価値）という言葉もよく聞きます。これを知ると何が嬉しいんですか？**
→ 狙い: 「1人の顧客が生涯でいくら使うか」がわかると、広告費の上限や投資判断ができるようになる
→ SEOキーワード: LTV 顧客生涯価値

**Q7. 「このお客さん、もうすぐ離れそう」みたいな予測もできるんですか？**
→ 狙い: チャーン予測・離脱予兆分析の話。経営者にとって「守りのデータ活用」としてインパクトが大きい

### パート3: 導入のリアル（費用・期間・体制）

**Q8. ぶっちゃけ、CDPの構築ってどのくらいお金がかかりますか？**
→ 狙い: SaaS型CDP（月数十万〜数百万）vs BigQuery型（構築費+ランニング安い）の比較。中小企業にはBigQuery型が現実的
→ SEOキーワード: CDP構築 費用

**Q9. 社内にデータの専門家がいない場合、どうすればいいですか？**
→ 狙い: 「全部外注」ではなく「伴走型で内製化を目指す」Beekleのアプローチを自然に訴求

**Q10. データ活用を始めるとき、最初にやるべきことは何ですか？いきなり全部やろうとしない方がいいですか？**
→ 狙い: 段階的アプローチの重要性。まずGA4+BigQueryで始めて、効果が見えたら拡大する
→ Beekleのゼロスタートの考え方とも接続

### パート4: 実例・ストーリー（リアリティを出す）

**Q11. 今まで関わった案件で、「データを見たら意外なことがわかった」みたいなエピソードはありますか？**
→ 狙い: 具体的な発見のストーリーで読者の興味を引く。「直感と違った」系の話が響く

**Q12. データ分析の結果を経営者に伝えるとき、どんな伝え方をすると刺さりますか？**
→ 狙い: 「数字の羅列ではなくストーリーで伝える」という話。ダッシュボードの見せ方なども

**Q13. 逆に、「データ活用がうまくいかなかった」パターンってありますか？何が原因でした？**
→ 狙い: 失敗談から学ぶ系コンテンツ。「基盤を作ったが誰も使わなかった」「目的なくデータを集めた」など

### パート5: 未来の話（2025-2026年のトレンド）

**Q14. 生成AIの登場で、データ分析の世界はどう変わりましたか？経営者にとって何が変わりますか？**
→ 狙い: 「AIに質問するだけでデータから答えが返ってくる時代」の到来。Text-to-SQLなどの最新動向

**Q15. 5年後、データを活用している企業とそうでない企業で、どのくらい差がつくと思いますか？**
→ 狙い: 経営者の危機感に訴えかける。「今始めないと取り残される」メッセージ

## 収録Tips

- 専門用語が出たら必ず「経営者にわかる言葉で言うと？」と聞き返す
- 具体的な数字（〇%改善、〇万円削減）を引き出す
- 「御社のクライアントで言うと...」と実例ベースで話してもらう
- 佐藤さんの経歴（京大数学→アクセンチュア→フリーランス）を活かした信頼性を出す
