# ADR 0001: コラム本数の目安を「100本」に置く

- Status: Accepted
- Date: 2026-05-03
- Deciders: kunio (Beekle)

## Context

- 2026-05-03 時点で公開済みコラムは 63 本（`src/data/column-embeddings.json` の records 数）、`claudedocs/drafts/` に下書き 38 本。
- 直近90日のSC実績（`claudedocs/content-marketing-strategy-2026-05.md` §1）は表示 ~1,100/月・クリック ~48/月・インデックス率 28.4%。クラスター単位の勝ち負けを語れる母数になっていない。
- 「効くまでに何本必要か」という問いに対して、SEOコンテンツ投資の経験則上、コーポレートサイトでクラスター戦略を評価可能な最低ロットは概ね100本前後（クラスターあたり10〜15本 × 6〜8クラスター）。
- これより少ない本数でCTR/順位/CVを語ると、サンプル不足で「効かない」と早期に判断してしまい、量産フェーズで方針を頻繁に揺らすリスクがある。

## Decision

**コラム本数の最低目安を「公開100本」と置く。100本到達までは、量産方針そのものを見直さない。**

具体的には:

1. 100本到達までは「コンテンツマーケが効かない」「クラスター戦略が間違っている」という結論を出さない。短期KPI（月次クリック/順位）が下振れしても、戦略レイヤ（クラスター構成・ターゲットKW）は変えない。
2. 個別記事の品質改善・タイトル書き換え・内部リンク補強は随時行う（戦略変更ではなく執行改善）。
3. 100本到達時点で、クラスター単位の表示/クリック/順位を評価し、勝ちクラスターへの集中投資 or 撤退判断を行う（次ADRの起点）。
4. 「100本」は粗い目安であり厳密な閾値ではない。±10本程度の前後で評価のタイミングは判断する。
5. **50本到達以降は「新規量産だけでなく、既存資産の手当ても並行して進める」**。具体的には:
   - **順位が付いていない既存記事のリライト**: SC で表示はあるが順位30位以下、または表示自体が極小の記事をタイトル・導入・H2構成・内部リンクで手当てする。
   - **既存記事からの切り出し記事化**: 1本に詰め込みすぎているテーマを、独立した深掘り記事として分割・新規化する（クラスター内の網羅性向上＋内部リンクハブ化）。
   - 配分の目安は「新規:リライト/切り出し = 概ね 7:3 〜 6:4」。100本の母数カウントには新規・切り出しを含めるが、純粋なリライト（同一URLの改稿）は含めない。

## Consequences

### Positive

- 戦略を頻繁に揺らさず、量産に集中できる（早期撤退バイアスの抑制）。
- 「あと何本」で進捗を語れるため、執筆の優先順位付けが単純化する（残り37本 → 月10本×4ヶ月 等の逆算が可能）。
- 100本までの間は、施策評価より「書ききる」執行に集中できる。

### Negative / リスク

- 明らかに間違った方向（例: ターゲットKWが完全に外れている）でも、本数到達まで惰性で書き続ける危険。→ §Decision-1 の対象は「戦略レイヤ」、§Decision-2 の執行改善は止めないことで緩和。
- 100本という数値そのものが恣意的。実証済みではなく経験則。

### 評価の停止条件（このADRを再検討するトリガ）

以下のいずれかが起きた場合、100本を待たずに本ADRを再評価する:

- 90日連続でインデックス率が30%を割り続ける（量を増やしても載らない構造的問題の可能性）。
- 単一クラスター（例: 要件定義/上流）で10本以上書いてもクリックがゼロのまま3ヶ月経過。
- ビジネス側でコンテンツマーケ予算自体の打ち切りが決まった場合。
